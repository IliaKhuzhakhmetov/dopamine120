## Getting Started

Fetch dependencies in the package you are working on:

```sh
cd packages/dopamine_ui
flutter pub get
```

Run the Widgetbook catalog:

```sh
cd packages/dopamine_ui/example
flutter pub get
flutter run 
```

## Development Commands

Run these from `packages/dopamine_ui` unless noted otherwise:

```sh
dart format .
flutter analyze
flutter test
```
