# DOPAMINE120

DOPAMINE120 is a Flutter monorepo. The active package is the reusable
`dopamine_ui` design system, with a Widgetbook catalog for browsing and testing
components during development.

## Repository Layout

- `packages/dopamine_ui/` - UI kit with design tokens, theme setup, and core
  `Dop*` widgets.
- `packages/dopamine_ui/example/` - Widgetbook catalog app for the UI kit.
- `apps/dopamine_app/` - placeholder for the future product app.
